# Sales Anomaly Detection for a Multi-Outlet F&B Business

This repository contains the code and data for detecting sales anomalies in a multi-outlet food and beverage business. The project is structured as follows:

```
/
├── data/
│   ├── raw/
│   │   ├── Transactions.csv
│   │   └── Products.csv
│   ├── processed/
│
├── notebooks/
│   ├── 01_data_understanding.ipynb
│
├── reports/
│   ├── figures/
│
├── README.md
├── requirements.txt
└── .gitignore
```

## Getting Started

1. Place the raw data files (`Transactions.csv` and `Products.csv`) in the `data/raw/` directory.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Open the Jupyter notebooks in the `notebooks/` directory to explore and analyze the data.

## Project Structure

- **data/**: Contains raw and processed data.
- **notebooks/**: Jupyter notebooks for data exploration and analysis.
- **reports/**: Generated figures and reports.
- **README.md**: Project documentation.
- **requirements.txt**: List of Python dependencies.
- **.gitignore**: Specifies intentionally untracked files to ignore.
